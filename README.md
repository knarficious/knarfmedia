A Blog CMS

### Technologies
Project is created with:
* PHP: 8.1.6
* Symfony: 5.4.20
* MySQL|PostGreSQL
* Webpack
* Javascript
* JQuery
* Bootstrap 5

### Setup

- Clone the Github repository https://github.com/knarficious/knarfmedia.git
- Install PHP dependencies with Composer
- Build with Yarn
