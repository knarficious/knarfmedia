A Blog CMS

### Technologies
Project is created with:
* PHP: 8.0.11
* Symfony: 5.3.13
* MySQL|PostGreSQL
* Node.js
* Webpack
* Javascript
* JQuery
* Bootstrap 5

### Setup

- Clone the Github repository https://github.com/knarficious/knarfmedia.git
- Install PHP dependencies with Composer
- Build with Yarn
